# Workplace Resource Guide

A comprehensive web-based guide to understanding and developing professionalism in the workplace.

## Features

- **Interactive Tabbed Interface**: Navigate through 6 key areas of workplace professionalism
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Clean, Modern UI**: Professional appearance with smooth animations
- **Comprehensive Content**: Covers appearance, communication, behavior, time management, and ethics

## Sections

1. **What Is Professionalism?** - Foundation concepts and core definitions
2. **Appearance & Dress Code** - Professional presentation and standards
3. **Communication Skills** - Verbal, written, and interpersonal communication
4. **Workplace Behavior** - Daily practices and team collaboration
5. **Time Management** - Productivity and organizational skills
6. **Ethics & Integrity** - Values, accountability, and professional character

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- Google Fonts (DM Serif Display, DM Sans)

## Usage

Simply open `index.html` in any modern web browser. No server required for local viewing.

- Chrome 70+
- Firefox 65+
- Safari 12+
